# Accenture Holidays for bangalore - 2025 Year

# ---------------------------------------------------

# To see entire year(2025) holidays (Floating / Mandatory)

https://localhost/holiday

# To see specific month holidays (For ex:- December month)

https://localhost/holiday?month=12

# To see mandatory holidays for whole year

https://localhost/holiday?htype=Mandatory

# To see Floating holidays for whole year

https://localhost/holiday?htype=Floating

# To see Mandatory Holidays for specific month (October)

https://localhost/holiday?htype=Mandatory&month=10

# To see Floating Holidays for specific month (February)

https://localhost/holiday?htype=Floating&month=2
